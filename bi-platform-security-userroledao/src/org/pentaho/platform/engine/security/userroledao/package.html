Defines a data access object (DAO) for create, retrieve, update, and delete (CRUD) operations on users and roles. This DAO is needed in two places: (1) the Pentaho User Console for changing users and roles and (2) the Pentaho BI Platform for authentication. One implementation of IUserRoleDao is based on Hibernate. It uses Spring's HibernateDaoSupport.
